# WordCloud Studio - 高性能中文词云生成器

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![PySide6](https://img.shields.io/badge/GUI-PySide6-green)
![License](https://img.shields.io/badge/license-MIT-orange)

**WordCloud Studio** 是一款基于 Python (PySide6) 开发的现代化桌面应用，专为中文文本分析和词云可视化设计。它集成了高性能分词算法、智能实体识别（人名/地名）、多方案配置管理以及强大的图形渲染引擎，旨在提供“苹果级”的用户体验。

## ✨ 核心特性 (Features)

* **🎨 现代化 UI 设计**：遵循 Apple HIG 设计规范，拥有清爽的卡片式布局、精致的交互动画和自适应的界面结构。
* **⚡ 高性能分词**：内置多进程并行处理引擎，利用多核 CPU 加速文本分词，GB 级文本也能快速处理。
* **🧠 智能实体提取**：除了基础分词，还支持**仅提取人名**、**仅提取地名**或两者混合模式，精准挖掘文本核心要素。
* **🛠️ 深度配置管理**：
    * **多方案切换**：可保存多套配置（如“小说分析”、“财报分析”），一键切换保留词和停用词表。
    * **词典干预**：内置可视化的强制保留词和停用词编辑器。
* **🖼️ 强大的蒙版系统**：
    * **内置素材库**：自带分类明确的形状素材（基础图形、动物、植物等）。
    * **智能识别**：支持透明 PNG 和白底黑图的自动识别与抠图。
* **📊 双视图模式**：
    * **词云视图**：支持最高 **8K** 分辨率渲染，支持透明背景（PNG Alpha 通道）。
    * **数据视图**：提供词频统计表格，支持实时模糊搜索、一键屏蔽/恢复词语，并可导出为 Excel (CSV)。
